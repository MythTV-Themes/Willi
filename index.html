Willi Theme for MythTV by managementboy

Willi is a 16:10 theme for MythTV for screens optimized for a resolution 1680x1050. First published 04-12-2010 and still actively mantained. If you want to help, please contact me.

Screenshot? http://www.mythtv.org/wiki/File:Myth-screenshot-2010-04-12T22-47-06.393.jpg

Dependencies

MythTV

Install

download it to /usr/share/mythtv/themes or select it from the theme selection screen (0.25 or newer)
